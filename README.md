## About Laravel

Laravel with Docker setup for web application framework with expressive, elegant syntax. We believe development must be an enjoyable and creative experience to be truly fulfilling. Laravel takes the pain out of development by easing common tasks used in many web projects, such as:


### Docker list containers
docker ps

### pull an image or a repository from a register
docker pull

### Push an image or a repository to an registry
docker push
### Rename a container
docker rename

### Stop All running container in docker
docker kill $(docker ps -q)

### Remove All Docker Containers
docker rm $(docker ps -a -q)

### Remove All Docker Images
docker rmi $(docker images -q)
